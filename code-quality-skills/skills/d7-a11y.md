You are sub-skill **D7 A11y**. Standard contract.

## AUDIT

- **Interactive divs**: `rg -n "<div[^>]*onClick" src/` — list every hit. For each, verify `role`, `tabIndex`, and a keyboard handler (`onKeyDown`/`onKeyUp` for Enter/Space).
- **Focus rings**: spot-check `Button`, `Tabs`, `Tab`, `IconButton` style files for `focus-visible:` / `outline:` / `ring-` Tailwind classes.
- **Image alt-text**: `rg -n "<img\s" src/` and `<Image ` — all should have `alt` (empty `alt=""` is OK for decorative).
- **Form labels**: `rg -n "<input " src/` — each should have an `id` referenced by a `<label htmlFor=>` OR an `aria-label`/`aria-labelledby`.
- **Heading order**: spot-check 3 page templates for heading hierarchy (no `<h3>` before `<h2>`).
- **Colour contrast**: pick 5 representative `bg-* text-*` Tailwind combos and report estimated contrast ratio. (If `axe-core` is available, run it on the dev server for one route and copy the JSON.)
- **Live-regions**: in chat / toast / status code, look for `aria-live` on dynamic regions.
- **`prefers-reduced-motion`**: search for `@media (prefers-reduced-motion` or `useReducedMotion` hook usage; flag pages with heavy animations and no opt-out.

For every file you read, append `"d7-a11y"` to `_file-coverage.json` `reviewed_by[]`.
