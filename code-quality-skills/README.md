# code-quality-skills/ — derivable cache

This folder is **regenerated** on every run from `code-quality-checker.md` at the repo root.
Do not edit files here directly — edit the master `code-quality-checker.md` and re-run with `--refresh-kit`.

The master file contains:
- § Shared contract — what every sub-agent does
- § Skill kit — embedded D1..D15 templates
- § Output formats — REPORT.md, REPORT.compact.txt, dashboard.html, fix-prompts.md

Run output goes to `audit-reports/<run-id>/`.
