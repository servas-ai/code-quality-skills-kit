Quality
d1 Correctness & Invariants w15 ▶

Catches the bugs that ship.

  • Race conditions in async flows
  • Swallowed catches (empty catch {})
  • as any escape hatches
  • Broken project-defined invariants

Validates that tests actually run and cover changes.

  • Pass / fail counts from real runs
  • .skip / .todo / .only inventory
  • Missing tests for changed modules
testsci
d6 Architecture w10 ▶

Keeps the codebase maintainable as it grows.

  • God-files (>300 LOC)
  • Layer violations
  • Circular dependencies
  • Barrel-file hygiene
invariants
d8 Dead code & Tech debt w8 ▶

Surfaces what should be deleted.

  • Unused exports across the graph
  • Stale TODOs (>30 days)
  • Placeholder files / scaffold leftovers
dead-code
Types & Build
d2 Type Safety w12 ▶

Trust the compiler, then check what it can’t.

  • Real tsc error counts (not estimates)
  • Missing strict flags
  • any inventory
  • @ts-ignore debt
types

What does CI actually say right now?

  • Exit codes for typecheck / test / build / lint
  • GitHub Actions success rate (last N runs)
ci

  • pnpm audit / equivalent
  • Outdated & duplicate packages
  • License risk surface

  • useEffect heat map

  • Unscoped imports & missing memoization

  • Scope drift between README & code

  • Hardcoded UI strings escaping the catalog

  • Dark-mode parity gaps

Get started

Install, configure, and watch it run

Three install paths. Four config questions. One audit you can replay below.

Install 3 ways 

curl -sSL https://raw.githubusercontent.com/servas-ai/code-quality-skills-kit/main/install.sh | sh
git clone --depth 1 https://github.com/servas-ai/code-quality-skills-kit /tmp/cqsk
cp /tmp/cqsk/code-quality-checker.md ./
cp -r /tmp/cqsk/code-quality-skills ./
echo "audit-reports/" >> .gitignore
# Drop into ~/.claude/commands/ for global slash command
mkdir -p ~/.claude/commands
curl -sSL https://raw.githubusercontent.com/servas-ai/code-quality-skills-kit/main/code-quality-checker.md \
  -o ~/.claude/commands/code-quality-checker.md
